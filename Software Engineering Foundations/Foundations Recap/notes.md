# Foundations Recap

## Spaceship operator

* Spaceship operator ```<=>``` is used to compare two values. It combines ```>, < , >=, <=```.

```ruby
a <=> b

# returns -1 if a < b 
# returns 0 if a == b
# returns 1 if a > b
```

* Let's see it in action

```ruby
1 <=> 2 # -1
2 <=> 2 #  0
3 <=> 2 #  1
```
